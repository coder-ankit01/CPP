# C++ Fundamentals

- Switch Case
- Loops (For, While, Do-While)
- Break and Continue

### Functions
- Function Declaration
- Function Definition
- Function Parameters
- Return Values
- Scope of Variables

### Arrays
- One-Dimensional Arrays
- Two-Dimensional Arrays
- Array Traversal
- Basic Array Problems

### Strings
- Character Arrays
- String Operations
- String Functions

### Pointers
- Pointer Basics
- Pointer Arithmetic
- Passing Pointers to Functions

### Recursion
- Basic Recursion
- Recursive Problem Solving

### Basic Mathematics
- Prime Numbers
- Factorial
- Fibonacci Series
- GCD & LCM
- Power Function

### Pattern Problems
- Star Patterns
- Number Patterns
- Pyramid Patterns

## Technologies Used
- C++
- GCC Compiler
- Visual Studio Code

## Purpose
- Learn C++ from scratch
- Build programming fundamentals
- Prepare for Data Structures and Algorithms
- Strengthen problem-solving skills
